# python-tools

Verzameling van Python desktop tools voor Windows 11.

## Apps

| App | Omschrijving |
|-----|-------------|
| transcribe | Lokale audio/video transcriptie met AI (faster-whisper + pyannote) |
| audioforge | Audio converter en editor met waveform visualisatie |

## Releases

Collega's downloaden de kant-en-klare .exe via GitHub Releases.  
Geen Python-installatie vereist.
